Automate your development workflow with multi-agent AI collaboration

Multi-Agent System

Team Lead analyzes tasks, Developer implements them. Automated collaboration between AI agents.

Task Queue Management

Automatic task discovery, prioritization, and tracking. Never lose track of what needs to be done.

🔄

Checkpoint Recovery

Stop and resume workflows anytime. Your progress is always saved.

📈

Real-time Monitoring

Watch progress in real-time with detailed logs and status updates.

Installation

Prerequisites

  • Node.js >= 18.0.0
  • Claude Code CLI installed and authenticated

Global Installation (Recommended)

npm install -g @graygate/claude-orchestrator

Run with npx

npx @graygate/claude-orchestrator

Verify Installation

claude-orchestrator --version

Quick Start

1

Initialize Your Project

Navigate to your project directory and initialize the orchestrator.

cd your-project
claude-orchestrator init
2

Start Development

Define your development scope and goals to begin automated development.

claude-orchestrator start \
  --scope "Implement user authentication" \
  --goals "Login,Register,Password Reset"
3

Monitor Progress

Watch the agents work and monitor the progress in real-time.

# View current status
claude-orchestrator status

# Watch logs in real-time
claude-orchestrator logs --follow

Commands Reference

init

Initialize the orchestrator in a project directory.

claude-orchestrator init [directory]

Arguments

  • [directory] - Target project directory (default: current)

start

Start the orchestration process.

claude-orchestrator start [options]

Options

  • --scope <scope> - Development scope description
  • --goals <goals> - Comma-separated development goals
  • --max-tasks <n> - Maximum tasks per cycle (default: 10)
  • --platform <platform> - Target platform (android, ios, web, custom)
  • --continuous - Enable continuous discovery mode
  • --dry-run - Preview without execution
  • --skip-permissions - Skip Claude permission prompts

status

Show current orchestration status.

claude-orchestrator status

Displays

  • Active agents and their status
  • Current task being processed
  • Queue statistics
  • Recent activity

logs

View development logs.

claude-orchestrator logs [options]

Options

  • --tail <n> - Show last n entries
  • --follow - Stream new log entries
  • --task <id> - Filter by task ID

stop

Stop running agents gracefully.

claude-orchestrator stop

Actions

  • Sends graceful shutdown signal
  • Saves current state for later resume
  • Marks in-progress tasks appropriately

resume

Resume from the last checkpoint.

claude-orchestrator resume

Actions

  • Loads saved state from previous run
  • Continues processing in-progress tasks
  • Restarts agent processes

Interactive Mode Shortcuts

During orchestration, use these keyboard shortcuts:

Key Action
l Show recent log entries
s Show current status
p Pause after current task
q Stop gracefully
r Resume paused work

How It Works

Claude Orchestrator coordinates two AI agents - a Team Lead and a Developer - to automate your software development workflow.

The Team Lead analyzes tasks and creates detailed implementation instructions, while the Developer executes them with precision.

1

Discovery (Optional)

  • Scan project structure
  • Identify missing features
  • Create tasks in queue
↓
2

Team Lead Analysis

  • Analyze task requirements
  • Study reference code
  • Write implementation guide
↓
3

Developer Implementation

  • Read instructions
  • Implement features
  • Run verification
  • Submit completion report
↓
4

Review

  • Verify implementation
  • Check build status
  • Log completion/rejection

Task States

pending → in_progress → awaiting_review → completed
in_progress → rejected

Configuration

The orchestrator creates a .claude-orchestrator/ directory in your project:

.claude-orchestrator/
├── config.json        # Project configuration
├── status.json        # Current agent status
├── queue.json         # Task queue
├── messages/          # Inter-agent communication
│   ├── to-developer.json
│   └── to-team-lead.json
└── logs/
    └── log.md         # Development log

config.json Example

{
  "project": {
    "name": "your-project",
    "path": "/absolute/path/to/project"
  },
  "scope": "Development scope description",
  "goals": ["goal1", "goal2"],
  "platform": "web",
  "maxTasks": 10,
  "continuous": false
}

Configuration Options

Option Type Description
project.name string Project name (auto-detected)
project.path string Absolute path to project
scope string Development scope description
goals string[] List of development goals
platform string Target platform: web, android, ios, custom
maxTasks number Maximum tasks per cycle (1-50)
continuous boolean Enable continuous discovery mode

Troubleshooting

Claude Code Not Found

If you see "Claude Code CLI not found":

npm install -g @anthropic-ai/claude-code

Authentication Required

If Claude Code is not authenticated:

claude login

Task Stuck

If a task appears stuck:

# Check status
claude-orchestrator status

# View recent logs
claude-orchestrator logs --tail 50

# Stop and resume
claude-orchestrator stop
claude-orchestrator resume
